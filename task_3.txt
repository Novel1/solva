IP-адрес — это уникальный идентификатор, присвоенный каждому устройству, подключенному к сети. Он позволяет устройствам
обмениваться информацией, определяя их местоположение и обеспечивая доступность для связи. IP-адрес состоит из строк чисел,
разделенных точками. Пример IPv4-адреса: 192.137.1.107. В IPv4 адресе используется четыре числа, каждое из которых может
быть в диапазоне от 0 до 255.

IP адреса начинают кончаться, из-за роста количества устройств, поэтому был разработан IPv6. У него более длинные адреса.
Чтобы упростить работу пользователей, IP-адреса связывают с доменными именами с помощью DNS.
Это позволяет использовать легко запоминаемые имена (например, www.google.com) вместо числовых IP-адресов при доступе к веб-сайтам.

TCP и UDP
1) Наверное одно из самых главных отличий, это надежность.
TCP гарантирует доставку данных. UDP не гарантирует.
Например: TCP используется для фото, а UDP для онлайн эфира.
2) Упорядочность доставки данных.
TCP обеспечивает порядок доставки данных
UDP не гарантирует
3) Скорость у UDP выше чем у TCP.
4) TCP использует больше ресурсов для управлением соединения и надежности. UDP использует меньше ресурсов.
5) Соединение
TCP устанавливает соеденение между отправителем и получателем. UPD нет.
